# 📋 Sira AI Gateway - 项目详细描述

## 🌟 项目概述

**Sira AI Gateway** 是一款专为现代企业打造的智能API网关系统，专注于解决AI服务集成、性能优化和成本控制的核心问题。通过先进的路由算法和多层缓存技术，为AI应用提供高性能、高可用、高安全的统一服务入口。

### 核心组件

#### 1. 智能路由引擎 (Intelligent Routing Engine)
- **复杂度分析**: 基于请求内容复杂度评估
- **性能监控**: 实时监控各服务商响应时间
- **成本计算**: 动态计算各模型的实际成本
- **决策算法**: 多维度权重决策路由选择

#### 2. 多层缓存系统 (Multi-layer Cache System)
- **L1缓存**: 内存缓存，最快访问速度
- **L2缓存**: Redis分布式缓存，高可用
- **L3缓存**: 文件系统缓存，大数据量存储
- **智能失效**: 基于访问频率的缓存策略

#### 3. 安全认证中心 (Security & Authentication)
- **多认证方式**: JWT、API Key、OAuth2
- **权限控制**: 基于角色的访问控制(RBAC)
- **安全审计**: 完整的操作日志和审计追踪
- **加密传输**: HTTPS强制加密传输

#### 4. 监控告警系统 (Monitoring & Alerting)
- **系统指标**: CPU、内存、网络、磁盘监控
- **业务指标**: 请求量、成功率、成本、用户行为
- **告警规则**: 可配置的告警阈值和通知方式
- **可视化面板**: Grafana集成监控大屏

### 技术栈

#### 后端技术栈
- **运行时**: Node.js 18+ (LTS)
- **Web框架**: Express.js 4.x
- **缓存系统**: Redis 7.x + 内存缓存
- **数据库**: PostgreSQL 15+ (可选)
- **消息队列**: NATS (可选)

#### 前端技术栈
- **管理面板**: HTML5 + CSS3 + JavaScript
- **图表库**: Chart.js (轻量级图表)
- **响应式设计**: 适配桌面和移动设备

#### DevOps技术栈
- **容器化**: Docker + Docker Compose
- **编排**: Kubernetes (可选)
- **CI/CD**: GitHub Actions
- **监控**: Prometheus + Grafana

## 🔄 部署方式

### 快速启动 (Docker)
```bash
# 使用Docker Compose启动
docker-compose -f docker-compose.simple.yml up -d

# 访问管理面板
open http://localhost:8080/admin
```

### 生产部署 (Kubernetes)
```bash
# 部署到Kubernetes集群
kubectl apply -f k8s/

# 配置Ingress和LoadBalancer
kubectl apply -f k8s/ingress.yml
```

### 云服务部署
- **AWS**: 支持EC2、ECS、EKS部署
- **阿里云**: 支持ECS、ACK部署
- **腾讯云**: 支持CVM、TKE部署
