# 🤝 行为准则 (Code of Conduct)

## 📋 我们的承诺

我们致力于为所有人提供一个无骚扰的社区环境，不论年龄、体型、残疾、民族、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族或宗教。

我们致力于营造一个开放、包容和多元化的社区环境。

## 🎯 我们的标准

### ✅ 鼓励的行为

**促进积极社区环境，包括但不限于：**

- 使用友好和包容性的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同情

### ❌ 不被接受的行为

**以下行为是不被接受的：**

- 使用性化语言或图像，以及任何形式的性骚扰
- 进行人身攻击或政治攻击（trolling）
- 公开或私下骚扰他人
- 未经明确许可发布他人的私人信息，如地址或电子邮件
- 其他有理由认为不适当的行为

## 📋 责任与后果

### 社区维护者的责任

项目维护者有责任澄清和执行可接受行为的标准，并对任何不可接受行为采取适当和公平的纠正措施。

项目维护者有权和义务删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki编辑、问题和其他贡献，并会在适当时候说明采取措施的理由。

### 执行指南

社区维护者将遵循这些社区影响指南来确定任何违反本行为准则行为的后果：

#### 1. 警告 (Correction)

**社区影响**: 使用不恰当的语言，或在社区中表现得其他不当行为。

**后果**: 来自社区维护者的私人书面警告，明确违规性质，并解释为什么这种行为是不适当的。可能需要公开道歉。

#### 2. 警告 (Warning)

**社区影响**: 通过单一事件或一系列行为造成的违规。

**后果**: 警告并附带违规后果。在指定时间内不得与相关人员互动，包括主动与执行行为准则的社区成员互动。这包括避免在社区空间以及外部渠道（如社交媒体）进行互动。违反这些条款可能导致临时或永久禁言。

#### 3. 临时禁言 (Temporary Ban)

**社区影响**: 严重违反社区标准，包括持续的不当行为。

**后果**: 在指定时间内禁止与社区的任何互动或公开交流。在此期间，不得与相关人员公开或私下互动。违反这些条款可能导致永久禁言。

#### 4. 永久禁言 (Permanent Ban)

**社区影响**: 展示违反社区标准的模式，包括持续的不当行为、对个人进行骚扰，或对个人或群体进行攻击或贬低。

**后果**: 永久禁止在社区进行任何形式的公开互动。

## 🔍 申诉流程

### 如何报告违规行为

如果您遇到或目睹违反本行为准则的行为，请通过以下方式报告：

1. **发送邮件** 到 conduct@sira-project.com
2. **在GitHub上创建问题** 并标记为 `conduct-violation`
3. **联系项目维护者** 通过项目的联系方式

所有举报都将被保密处理。

### 申诉处理

1. **接收举报**: 维护者将在24小时内确认收到举报
2. **调查评估**: 在3-5个工作日内完成初步调查
3. **采取行动**: 根据调查结果采取适当措施
4. **反馈通知**: 向举报人和被举报人提供处理结果

### 申诉权利

任何被采取纠正措施的人都有权：
- 了解采取措施的具体原因
- 对决定提出上诉
- 要求匿名处理（在合理范围内）

## 🎯 适用范围

本行为准则适用于：
- 项目仓库及其所有子模块
- 项目相关的所有公共论坛和社区空间
- 项目相关的线下活动和会议
- 项目相关的社交媒体账户和渠道
