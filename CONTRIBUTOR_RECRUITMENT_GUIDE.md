## 🚀 如何开始贡献？

### Step 1: 熟悉项目
```bash
# 1. Fork项目到你的GitHub账户
# 2. 克隆到本地
git clone https://github.com/YOUR_USERNAME/sira-ai-gateway.git
cd sira-ai-gateway

# 3. 安装依赖
npm install

# 4. 启动开发服务器
npm start

# 5. 访问 http://localhost:8080 查看效果
```

### Step 2: 选择任务
访问我们的 [GitHub Issues](https://github.com/your-repo/sira-ai-gateway/issues) 页面：

**🟢 新手友好任务：**
- [good-first-issue](https://github.com/your-repo/sira-ai-gateway/labels/good%20first%20issue) - 适合新手的简单任务

**🟡 中级任务：**
- [enhancement](https://github.com/your-repo/sira-ai-gateway/labels/enhancement) - 功能增强
- [performance](https://github.com/your-repo/sira-ai-gateway/labels/performance) - 性能优化

**🟠 高级任务：**
- [architecture](https://github.com/your-repo/sira-ai-gateway/labels/architecture) - 架构改进

```bash

git checkout -b feature/amazing-contribution

- Add feature description

```

## 🎯 贡献指南

### 代码规范
```javascript
// ✅ 推荐写法
function processRequest(request) {
  if (!request) {
    throw new Error('Request is required');
  }

  return this.validateRequest(request)
    .then(() => this.processData(request.data))
    .catch(error => this.handleError(error));
}

// ❌ 避免的写法
function processrequest(req){if(!req)throw new Error('no req');return this.validaterequest(req).then(()=>this.processdata(req.data)).catch(e=>this.handleerror(e));}
```

### 提交规范
```
feat: add user authentication feature
fix: resolve memory leak in cache manager
docs: update API documentation for v2.1.0
style: format code with prettier
refactor: simplify routing logic
test: add unit tests for new feature
chore: update dependencies
```
